Grid

All columns class:
  • col-{{i+1}}
All gaps class:
  • gap-{{(i+1)*10}}

Responsive

All responsive columns class apply to the breakpoint and above:
  • Small: >= 320px
  • Medium: >= 420px
  • Large: >= 768px
  • Extra Large: >= 1440px
For example, col-1-md will apply to medium and above, col-1-lg will apply to large and above, and col-1-xl will apply to extra large and above. The class without breakpoint will apply to all screen, unless there is another class to ovewrite it.
