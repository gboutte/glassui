# Glassui

## Installation

```bash
npm install @glassui/glassui
```

Inside your `angular.json` file, add the following to the `styles` array:

```json
{
   "styles": [
       "node_modules/@gboutte/glassui/styles/theme.scss"
   ],
}
```
